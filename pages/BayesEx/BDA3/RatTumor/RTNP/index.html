Model:

The natural model here is a binomial model, with a beta prior on the probability of a tumours (θ \theta θ), so that we can exploit conjugacy if we want to. That is, we have the model:

y∼Binomial⁡(n,θ)y \sim \operatorname{Binomial}(n, \theta)y∼Binomial(n,θ) θ∼Beta⁡(α,β)\theta \sim \operatorname{Beta}(\alpha, \beta)θ∼Beta(α,β)

In this case, we have prior information in the form of 707070 previous experiments which use the same variant of rat. In these experiments, we expect the latent probability of a tumour to vary according to the experiment – it is probably a function of environment, diet, and so on.

We denote these previous experiments as (yi,ni)(y_i, n_i)(yi​,ni​), such that our current trial is (y71,n71)(y_{71}, n_{71})(y71​,n71​), corresponding to the (4,14)(4, 14)(4,14) scenario as originally outlined.

We can incorporate these other experiments into our model by estimating a unique θi\theta_iθi​ for each experiment, with the values of θi\theta_iθi​ drawn from a common prior over θ\thetaθ values. This extends the above model to:

yi∼Binomial⁡(ni,θi)y_i \sim \operatorname{Binomial}(n_i, \theta_i)yi​∼Binomial(ni​,θi​) θi∼Beta⁡(α,β)\theta_i \sim \operatorname{Beta}(\alpha, \beta)θi​∼Beta(α,β)

We need to specify a hyperprior over α\alphaα and β\betaβ –- BDA3 contains a long discussion on this issue. We choose to parametrize the mean and variance of the θ\thetaθ distribution, transforming to α\alphaα and β\betaβ after sampling.

Our NumPyro model here is:

def rat_mod_full(y=None, n=None):
    # Hyperpriors
    mu = numpyro.sample("mu", dist.Beta(2, 2))
    tau = numpyro.sample("tau", dist.Gamma(2, 0.5))

    # Transformation to alpha, beta
    alpha = mu * tau
    beta = (1-mu) * tau

    # Prior on the probability of a tumour
    with numpyro.plate("theta_plate", len(pop)):
        theta = numpyro.sample("theta", dist.Beta(alpha, beta))

    # Likelihood
    with numpyro.plate("obs", len(pop)):
        numpyro.sample("y", dist.Binomial(n, theta), obs = y)

Inference:

We use NUTS to sample from the posterior:

sampler = infer.MCMC(
    infer.NUTS(rat_mod_full),
    num_warmup=500,
    num_samples=1000,
    num_chains=1,
)

sampler.run(
    jax.random.PRNGKey(1),
    y=jnp.array(tumors),
    n=pop
)

samples = sampler.get_samples()
